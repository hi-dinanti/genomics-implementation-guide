Documentation for this format

This structure is derived from Extension

Summary

Complex Extension: Extension containing pricing information for charge item responses.

  • sequence: integer: Berisi data urutan terkait respon persetujuan tindakan dan layanan dengan tipe data positiveInt.
  • payor: Reference: Berisi data penjamin yang menyetujui tindakan dan layanan dengan tipe data Reference yang direferensikan ke data yang tersimpan di resource Organization.
  • identifier: string: Berisi data ID lokal terkait program yang dijalankan dengan tipe data Identifier.
  • coverage: Reference: Berisi data kepesertaan asuransi dengan tipe data Reference yang direferensikan ke data yang tersimpan di resource Coverage.
  • status: CodeableConcept: Berisi data yang menyatakan status persetujuan tindakan dan layanan dengan tipe data CodeableConcept.
  • unitPriceOverride: Money: Berisi data biaya yang disetujui per unit dengan tipe data Money.
  • factorOverride: decimal: Berisi data jumlah dari biaya yang disetujui per unit dengan tipe data decimal.
  • overrideReason: string: Berisi data untuk menyatakan keterangan maupun alasan dengan tipe data string.
  • quantity: decimal: Berisi data jumlah tindakan dan layanan yang disetujui dengan tipe data decimal.
  • total: Money: Berisi data total biaya yang disetujui dengan tipe data Money.
  • excess: Money: Berisi data nilai excess dengan tipe data Money.
  • unitPrice: Money: Unit price extension for ChargeItem.
  • totalPrice: Money: Total price extension for ChargeItem.
